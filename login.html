Sobre o Sistema

       

   A fiscalização do exercício da Odontologia é uma das atividades finalísticas dos Conselhos Regionais de Odontologia;


      Ao realizar esta atividade cada CRO armazena as dados de fiscalização em meio próprio;

 

    Com o objetivo de consolidar esses dados e gerar informações por meio de relatório e painéis o Conselho Federal de Odontologia desevolveu o Sistema FISCALIZA;


     No FISCALIZA cada CRO possui usuários com perfis para inserir os dados nessa base de dados coorporativa e outros com o perfil de gerar relatórios  como forma de subsidiar a decisão do Sistema Conselho de Odontologia.
